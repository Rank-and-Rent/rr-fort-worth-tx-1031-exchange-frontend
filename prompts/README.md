# Fort Worth 1031 Exchange Prompts

This directory contains prompts and data for generating 1031 exchange content for Fort Worth, TX.

## Structure

- **fort-worth-1031-prompts.json**: Contains 35 locations and 20 inventory types with generation prompts

## Locations (35)

The file includes 35 locations in and around Fort Worth, TX:
- Downtown Fort Worth
- Stockyards District
- Cultural District
- West 7th Street
- Near Southside
- TCU Area
- And 29 more surrounding areas

## Inventory Types (20)

20 different property types suitable for 1031 exchanges:
- Office Buildings
- Retail Properties
- Industrial Warehouses
- Multi-Family Apartments
- Single Tenant Net Lease
- Medical Office Buildings
- Self-Storage Facilities
- Hotels and Hospitality
- Mixed-Use Developments
- Land Development
- Student Housing
- Senior Living Facilities
- Manufacturing Facilities
- Distribution Centers
- Shopping Centers
- Strip Malls
- Restaurant Properties
- Automotive Service Centers
- Gas Stations and Convenience Stores
- Agricultural Land

## Usage

The JSON file contains:
1. **locations**: Array of 35 location names
2. **inventoryTypes**: Array of 20 property types
3. **promptTemplates**: Reusable templates for generating content
4. **generationPrompts**: Pre-built prompts for each location and inventory type

## Content Generation

Use the prompts in `generationPrompts` array to generate content for:
- Location-specific 1031 exchange guides
- Inventory type overviews
- Combined location + inventory type content
